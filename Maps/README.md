Basic Description
This project is SAP using Google Maps API to display a few National Parks in USA<br/>
The markers display the National Parks accross different states in the US<br/>
<br/>
<strong>Functionality of the page includes:</strong> <br/> 
	An interactive navigation bar to filter map and navigation bar results using knockout.js, <br/>
	Display of the nearest airport and local timezone using the SITA API<br/>
   Responsive design to be used and viewed on all web modern browsing devices.
